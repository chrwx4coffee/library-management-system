# KKU Kütüphane Otomasyonu (Library Management System)

Bu proje, C# ve WPF kullanılarak geliştirilmiş bir Kütüphane Yönetim Sistemidir.

## Özellikler

*   Kullanıcı Yönetimi (Ekleme, Silme, Güncelleme)
*   Kitap Yönetimi
*   Ödünç Alma ve Teslim Etme İşlemleri
*   Raporlama
*   SQLite Veritabanı Entegrasyonu
*   Entity Framework kullanımı

## Teknolojiler

*   C#
*   WPF (Windows Presentation Foundation)
*   .NET Framework 4.7.2
*   SQLite
*   Entity Framework 6

## Kurulum ve Çalıştırma

1.  Projeyi klonlayın (`git clone https://github.com/chrwx4coffee/library-management-system.git`).
2.  Visual Studio ile `KKU_Kutuphane.sln` dosyasını açın.
3.  NuGet paketlerini geri yükleyin.
4.  Projeyi derleyin ve çalıştırın.
